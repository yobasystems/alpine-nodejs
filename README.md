This Docker image [(yobasystems/alpine-nodejs)](https://hub.docker.com/r/yobasystems/alpine-nodejs/) is based on the minimal [Alpine Linux](http://alpinelinux.org/) with [Nodejs 6.6.0](https://nodejs.org/).


## Features

  * Minimal size only
    * 12.5 MB and only 2 layers
    * 11.1 MB and only 2 layers :min version
  * Memory usage is minimal on a simple install.
  * Node Version 6.6.0
  * NPM Version 3

## Creating an instance

    docker run yobasystems/alpine-nodejs

    or

    docker run yobasystems/alpine-nodejs:min
